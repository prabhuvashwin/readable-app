# Readable

React Nanodegree Project

## Installing server
  1. cd ~/readable-app/api-server
  2. npm install
  3. npm start
  Server will start running at port 3001

## Setting up client
  1. cd ~/readable-app/frontend
  2. npm install
  3. npm start
